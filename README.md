# 🏭 Warehouse Database

A warehouse inventory database project developed using **MySQL**. This project demonstrates the complete process of designing a relational database, including **Entity Relationship Diagram (ERD)**, **Database Normalization (up to 3NF)**, relational schema design, and SQL implementation.

---

## 📌 Project Overview

The purpose of this project is to design and implement a relational database for warehouse inventory management. The database manages products, warehouse areas, transportation activities, and warehouse transactions while maintaining data consistency through database normalization.

---

## ✨ Features

* Product Management
* Warehouse Area Management
* Transportation Management
* Warehouse Transaction Management
* Entity Relationship Diagram (ERD)
* Database Normalization (1NF – 3NF)
* Relational Schema Design
* Ready-to-run MySQL SQL Script

# 🚀 Getting Started

### Requirements

* MySQL Server 8.x or later
* MySQL Workbench (Recommended)

### Installation

1. Open MySQL Workbench.
2. Create a new SQL tab.
3. Copy the contents of:

```text
sql/warehouse_database.sql
```

4. Execute the SQL script.
5. The database is ready to use.
